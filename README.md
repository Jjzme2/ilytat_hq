# HQ.ILYTAT.com

The Operating System for ILYTAT LLC.

## Tech Stack
- **Framework**: Nuxt 3
- **Styling**: Tailwind CSS + CSS Variables (Dynamic Theming)
- **State Management**: Pinia
- **Backend**: Firebase (Firestore, Auth, Functions)

## Setup
1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

## Key Features
- **Dynamic Theming**: Switch between Light, Dark, and System preferences.
- **Responsive Layout**: Sidebar on desktop, Drawer on mobile.
- **Modular Architecture**: Feature flags via config (Coming Soon).
